The Basics

  • HTML stands for Hyper Text Mark-up Language, and is the basics code behind web pages.
  • In Internet Explorer load your favourite web page. Right click somewhere on the page, select view Source, you will now see the HTML code behind the page displayed.
  • You will be using software called Notepad++ to write your HTML.
  • An HTML file is just a page of text with lines of code set in tags. The tags are set in angled brackets < and >.
  • The tags tell the browser how to display the contents of the page.
  • Most tags come in pairs i.e. an opening and a closing tag.

for example

Opening Tag Closing Tag
<HTML> </HTML>
<TITLE> </TITLE>
<HEAD> </HEAD>
<BODY> </BODY>
  • One tag which does not need a closing tag is the <BR> tag, which is used to forces or break the text onto the next line, It's the HTML equivalent of hitting the Enter key on your keyboard when you type words in Microsoft Word.
  • If you'd like more help with the tags then you can go to https://www.w3schools.com/html/default.asp for details of all the tags and their attributes.
